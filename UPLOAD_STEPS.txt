✅ STEP 1: Create GitHub Account (5 minutes)

1. Go to: https://github.com
2. Click: "Sign up"
3. Enter email address
4. Click: "Continue"
5. Create password
6. Click: "Continue"
7. Enter username (e.g., "nafl-admin")
8. Click: "Continue"
9. Check email for verification
10. Click link in email
11. ✅ GitHub account created!

═════════════════════════════════════════════════════════════════════════════

✅ STEP 2: Create Repository (2 minutes)

1. Go to: https://github.com/new
2. Fill in:
   - Repository name: nafl-ptm
   - Description: Parent Teacher Meeting Schedule Portal
   - Visibility: PUBLIC (must be public!)
   - Initialize with: Nothing (leave empty)
3. Click: "Create repository"
4. ✅ Repository created!

═════════════════════════════════════════════════════════════════════════════

✅ STEP 3: Upload All Files (2 minutes)

1. You're now in your new repository (empty)
2. Click: "Add file" dropdown
3. Select: "Upload files"
4. Select ALL 14 files from the github-ready folder:
   ✓ index.html
   ✓ NAFL_PTM_Portal.html
   ✓ GitHub_README.md
   ✓ GITHUB_PAGES_SETUP.md
   ✓ LICENSE
   ✓ .gitignore
   ✓ COMPLETE_PACKAGE_README.md
   ✓ FINAL_AUDIT_REPORT.md
   ✓ DEPLOYMENT_CHECKLIST.md
   ✓ HOW_TO_DEPLOY.html
   ✓ DATA_SECURITY.md
   ✓ TEACHER_GUIDE.html
   ✓ GoogleAppsScript.gs
   ✓ README.md

   OR: Drag and drop all files at once!

5. Commit message: "Initial commit - NAFL PTM Portal"
6. Click: "Commit changes"
7. ✅ All files uploaded!

═════════════════════════════════════════════════════════════════════════════

✅ STEP 4: Enable GitHub Pages (1 minute)

1. You're in your repository
2. Click: Settings (top menu, right side)
3. Click: Pages (left sidebar)
4. Under "Build and deployment":
   - Source dropdown: select "Main"
   - Folder dropdown: select "(root)"
5. Click: Save
6. Wait 1-2 minutes...
7. ✅ You'll see: "Your site is live at: https://..."

═════════════════════════════════════════════════════════════════════════════

✅ STEP 5: Test Your Portal (1 minute)

1. Copy the URL shown in GitHub Settings → Pages
   (looks like: https://your-username.github.io/nafl-ptm/)

2. Open in browser

3. You should see: NAFL PTM Portal home page

4. Click: "Admin Portal"
